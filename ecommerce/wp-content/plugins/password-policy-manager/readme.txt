== Description ==

Password Policy Manager carring password management features like auto password expiration, one click password reset, Enforce strong password, role-based password policy, Enforce strong password for inactive users, password history mangement and many more. Password strength meter enables you to know the strength of your users' passwords then you can enforce password change, and you can force them to use strong passwords by configuring a strong password policy for them. In case of any suspicious activity, you can use a one-click password reset policy to suspend all the current sessions and send a reset password link, and force password change on all users. User password manager gives you full control of ways in which you can enhance your website's security.

= Enforce strong password =
On the first login user should reset the password and set new strong password.

= Force password change =
Administrators can enforce users to change their password on their next login.

= one click reset password =
reset password is an operation to invalidate the current password and force it to generate a new strong password. the administrator can reset all the passwords in case of any breach. This will kill all the current sessions and users will be forced to set a new strong password via email.

= Secure Password =
A password is a secure password if it carries a combination of small capital and special characters and a minimum length of 10.

= password security =
 For password security, a strong password should be longer than 8 characters and include some special characters and numeric digits, and it should be changed after a certain amount of time, which is supported by our password policy plugin.

= Enforce password change =
Enforce password change for users on their next login.

= Strong password =
Strong password: the strong password that both humans and computers can hardly detect, strong password carries all combination characters ($,#,0-9, etc.) with greater length.

= Password Score =
Password Score: password score will show all user's password strengths, you can check the users who are using a strong password, medium, and weak passwords, and based on that you can enforce them to use strong password according to password policy (like strong password, password expiration, one click reset password, enforce role based strong password, inactive user logout, password history management).

= Enforce strong password =
Enforce strong password: Enforce the users to set the password according to password policy (like strong password, password expiration, one click reset password, enforce role based strong password, inactive user logout, password history management) set.

== Installation ==

= From your WordPress dashboard =
1. Navigate to `Plugins > Add New` from your WP Admin dashboard.

2. Search for `password policy manager.

3. Install `password policy manager` and Activate the plugin.

= From WordPress.org =

1. Search for `password policy manager` and download it.

2. Unzip and upload the `password policy manager` directory to your `/wp-content/plugins/` directory.

3. Activate the password policy manager from the Plugins tab of your admin dashboard.

== Frequently Asked Questions ==

= I want to enable Password policy manager? =

You can Enable the Password policy Settings tab to enable the password policy features.

= I have enabled password policy settings for all users, what happens if an end-user tries to login? =

If a user has not logged in, the user has to login in site then it will be reset the password during the login. 

= I want to enable only password expiration for my users. What should I do? =

You can enable password expiration from password policy tab and then save the setting.

= Once Activated =

1. Select password policy manager from the left menu and follow the instructions.

2. You can start the Users' password policy manager and force password change.
